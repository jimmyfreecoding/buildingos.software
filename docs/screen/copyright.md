## 软件详细信息
- 硬件环境：
  - 大屏显示设备：LED 拼接屏/液晶拼接墙，分辨率≥4K，亮度≥500nit
  - 播放终端：工作站级主机，CPU 8核、RAM≥32GB、NVMe SSD≥1TB，独立显卡（支持 4K@60Hz 多路输出）
  - 采集与接口：HDMI/DP 输出，网口千兆，必要时视频采集卡
  - 数据接入设备：边缘网关（x86 四核、RAM≥8GB、SSD≥128GB，MQTT/HTTP）
  - 传感器与设备：温湿度、能耗计量（电/水/气/热表）、门禁/闸机、安防摄像机（1080p/H.265）
  - 服务器：x86 8核、RAM≥16GB、SSD≥512GB；或云主机同等规格
  - 网络与供电：千兆交换机、UPS≥1KVA、PoE（如需）
- 软件环境：
  - 开发操作系统：Linux（Ubuntu 22.04）
  - 开发环境/开发工具：Node.js 20.x、Vite、Vue 3、TypeScript 5.x、NestJS 10.x、pnpm/npm
  - 运行平台/操作系统：Linux（Ubuntu 22.04）
  - 运行支撑环境/支持软件：Nginx、PostgreSQL 15、Redis 7、MQTT Broker（EMQX/Mosquitto）、Docker、WebGL 3D 引擎
- 编程语言：TypeScript 5.x（前端/后端）、SQL（数据定义与查询）
- 源程序量：约31000行
- 开发目的：构建楼宇数字孪生大屏，统一展示与联动运营数据
- 面向领域/行业：智慧楼宇/园区可视化运维
- 软件主要功能：综合态势展示、设施设备可视化、综合安防联动、便捷通行数据展示、环境空间指标监测、能耗管理与分析、地图与楼层视图、主题与布局配置、多数据源接入与刷新策略、播放轮播
- 软件主要技术特点：应用软件；Vue3+NestJS+TypeScript 前后端分离；WebGL 3D 可视化；多源数据接入（MQTT/HTTP/DB）；边缘缓存与断点续传；权限审计与加密传输；规则化组件与场景布局
