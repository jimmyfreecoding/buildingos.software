## 使用说明

- 每个系统包含三页文档：
  - 软件著作权内容：用于申报信息的结构化填写
  - 使用手册：操作说明或设计说明模板（含章节结构）
  - 源代码：申报所需的“前30页”和“后30页”占位页
- 填写时请遵循各字段的格式要求与约束说明。使用手册中需补充操作截图或设计图；源代码页需粘贴规范化代码片段。
