## 软件详细信息
- 硬件环境：
  - 推理服务器：x86 16核，RAM≥64GB，NVMe SSD≥2TB；NVIDIA GPU≥24GB 显存（如 A5000/RTX 4090），千兆/万兆网口
  - 边缘设备（可选）：x86 四核，RAM≥8GB，SSD≥128GB，支持摄像头/麦克风接入
  - 多模态外设：摄像头（1080p/H.265）、麦克风阵列、扬声器，二维码/NFC 模块
  - 对象存储与向量检索：NAS/对象存储（MinIO/OSS），高可用存储阵列
  - 网络与安全：千兆/万兆交换机、企业级防火墙与 WAF、UPS≥1KVA
- 软件环境：
  - 开发操作系统：Linux（Ubuntu 22.04）
  - 开发环境/开发工具：Node.js 20.x、Vite、Vue 3、TypeScript 5.x、NestJS 10.x、pnpm/npm
  - 运行平台/操作系统：Linux（Ubuntu 22.04）
  - 运行支撑环境/支持软件：Nginx、PostgreSQL 15、Redis 7、MQTT Broker（EMQX/Mosquitto）、Docker、对象存储（MinIO/OSS）、向量数据库（Milvus/pgvector）、LLM 推理服务（vLLM/云端API）
- 编程语言：TypeScript 5.x（前端/后端）、SQL（数据定义与查询）

- 开发目的：提供多模态楼宇智能体，结合场景上下文与专用工具执行任务

- 软件主要功能：多模型接入与选择、图文音多模态理解、RAG 知识检索与上下文记忆、场景与会话历史提供上下文、工具编排（设备控制/告警/工单/预约）、任务审计与日志
- 软件主要技术特点：人工智能软件；LLM 智能体编排、函数调用与工具链、RAG/向量检索、会话记忆与多模态输入、权限审计与加密传输、边缘协同
