## 软件详细信息

### 硬件环境
- **软件开发硬件环境**：
  PC机：Intel Core i7或同等规格处理器，RAM≥16GB，硬盘≥512GB。
- **软件运行硬件环境**：
  门禁控制器：Cortex-A53 四核≥1.2GHz，RAM≥2GB，Flash≥16GB。
  边缘网关：x86 四核，RAM≥8GB，SSD≥128GB。
  服务器：x86 8核，RAM≥16GB，SSD≥512GB（或同等规格云主机）。
  配套设备：读卡器（Mifare/IC/NFC）、闸机（三辊/摆闸）、摄像机/人脸终端（1080p/H.265）、二维码扫码器。

### 软件环境
- **开发该软件的操作系统**：Windows 11 / macOS 14
- **软件开发环境/开发工具**：Node.js 20.x、Vite、Vue 3、NestJS 10.x、VS Code
- **该软件的运行平台/操作系统**：Linux（Ubuntu 22.04）/ Windows Server 2019
- **软件运行支撑环境/支持软件**：Nginx、PostgreSQL 15、Redis 7、MQTT Broker（EMQX/Mosquitto）、Docker

### 编程语言
- **编程语言**：TypeScript 5.x
- **源程序量**：3112

### 开发目的
- **内容**：实现访客与员工统一通行授权、门禁联动与安全审计，提升楼宇通行效率与安全性。

### 面向领域/行业
- **内容**：智慧楼宇、智慧园区门禁通行管理行业。

### 软件主要功能
- **内容**：
  本软件具备人员与访客全生命周期管理功能，支持预约审核及通行权限自动化下发；实现门禁设备联动控制与状态监测；支持二维码、人脸、NFC多模态核验；提供黑白名单管理、临时授权及异常告警处理；支持通行日志的全量查询、导出与审计，满足园区安防管理需求。

### 软件主要技术特点
- **分类**：人工智能软件 物联网软件 智慧城市软件
- **详细说明**：
  采用前后端分离架构（Vue3+NestJS+TS）与RBAC权限模型；基于MQTT/HTTP协议实现海量设备接入；具备边缘缓存与断点续传能力，确保数据完整性；通过统一审计与加密传输保障系统安全。
