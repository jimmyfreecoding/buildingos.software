## 软件详细信息
- 硬件环境：
  - 运维终端：PC/iPad/安卓手机；用于巡检与工单处理
  - 能耗计量：电/水/气/热表（RS-485/MBus），集中采集器
  - 传感器与设备：温湿度、功耗、振动、门磁等，RS-485/Zigbee/LoRa 接入
  - 报警与视频：报警主机、NVR/摄像机（1080p/H.265）
  - 边缘网关：x86 四核，RAM≥8GB，SSD≥128GB，千兆以太网，支持 MQTT/HTTP
  - 服务器：x86 8核，RAM≥16GB，SSD≥512GB；或云主机同等规格
  - 网络与供电：千兆交换机、UPS≥1KVA、PoE（如需）
- 软件环境：
  - 开发操作系统：Windows 11 / macOS 14
  - 开发环境/开发工具：Node.js 20.x、Vite、Vue 3、TypeScript 5.x、NestJS 10.x、pnpm/npm
  - 运行平台/操作系统：Linux（Ubuntu 22.04）
  - 运行支撑环境/支持软件：Nginx、PostgreSQL 15、Redis 7、MQTT Broker（EMQX/Mosquitto）、Docker
- 编程语言：TypeScript 5.x（前端/后端）、SQL（数据定义与查询）
- 源程序量：约33000行
- 开发目的：统一管理告警、巡检、工单与能耗，提升楼宇运营效率
- 面向领域/行业：智慧楼宇/园区运维管理
- 软件主要功能：设备告警与事件处理、设备台账与巡检计划、工单创建分派闭环、能耗数据采集与分析、运营看板与报表、消息通知与权限审计
- 软件主要技术特点：应用软件；Vue3+NestJS+TypeScript；RBAC 权限模型；时序数据与能耗分析；消息订阅与联动；边缘同步与加密传输
