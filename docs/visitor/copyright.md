## 软件详细信息

### 硬件环境
- **软件开发硬件环境**：
  PC机：Intel Core i7或同等规格处理器，RAM≥16GB，硬盘≥512GB。
- **软件运行硬件环境**：
  前台登记终端：一体机/PC（i5/8GB/256GB）、摄像头、二维码扫码器、热敏打印机（访客胸卡）。
  自助签到设备：安卓平板或嵌入式面板，支持二维码/人脸/NFC识读。
  门禁/闸机：三辊/摆闸，RS-485/干接点控制接口。
  人脸识别终端：1080p/H.265，红外补光，支持人脸核验。
  边缘网关：x86 四核、8GB RAM、SSD 128GB，千兆以太网，MQTT/HTTP。
  服务器：x86 8核、16GB RAM、SSD 512GB；或云主机同等规格。
  网络与供电：千兆交换机、UPS≥1KVA、PoE（如需）。

### 软件环境
- **开发该软件的操作系统**：Windows 11 / macOS 14
- **软件开发环境/开发工具**：Node.js 20.x、Vite、Vue 3、TypeScript 5.x、NestJS 10.x、pnpm/npm
- **该软件的运行平台/操作系统**：Linux（Ubuntu 22.04）
- **软件运行支撑环境/支持软件**：Nginx、PostgreSQL 15、Redis 7、MQTT Broker（EMQX/Mosquitto）、Docker

### 编程语言
- **编程语言**：TypeScript 5.x
- **源程序量**：约31000行

### 开发目的
- **内容**：规范访客预约与到访核验，联动通行控制。

### 面向领域/行业
- **内容**：智慧楼宇、园区访客管理行业。

### 软件主要功能
- **内容**：
  本软件实现访客全流程数字化管理，包括访客在线预约与审批、现场二维码/人脸/NFC多方式签到、身份核验与权限自动下发；支持黑白名单管理、异常通行告警及全量访问记录审计导出，提升楼宇安防等级与接待效率。

### 软件主要技术特点
- **分类**：人工智能软件 物联网软件 智慧城市软件
- **详细说明**：
  采用Vue3+NestJS前后端分离架构与RBAC权限模型；基于MQTT/HTTP协议接入多类型门禁设备；利用边缘缓存与断点续传技术保障数据可靠性；通过统一审计与加密传输确保访客数据安全。
